## 三、各元件接法細節

### 搖桿（2 顆）
- **左搖桿**：只用 上下(Y) → **GPIO34**（油門，已拆彈簧）；左右(X) 不用，空著。
- **右搖桿**：上下(Y) → **GPIO32**（pitch）、左右(X) → **GPIO33**（roll）；**彈簧留著**（放手回中）。
- 每顆搖桿：VCC → **3V3（不能 5V！）**、GND → GND、各軸抽頭 → 對應 ADC 腳。

### 模式開關 A / B（各一顆 MTS-103，3 腳）
```
上腳 ──────── 3V3
中腳(common) ─ GPIO（A=36 / B=39）
下腳 ──────── GND

節點(GPIO) 再加分壓定義中檔：
節點 ──[10kΩ]── 3V3
節點 ──[10kΩ]── GND
```
撥上=0、中(斷開)=1、下=2；**模式 = A×3 + B**（00=0 … 22=8）。

### 選單鈕電阻階梯（4 顆鈕共用 GPIO35）
```
3V3 ─┬─[＋鈕]──[1kΩ]───┐
     ├─[−鈕]──[4.7kΩ]──┤
     ├─[確定]──[10kΩ]───┼──→ GPIO35（節點）
     └─[返回]──[22kΩ]───┘
                         │
                       [10kΩ]   ← 下拉，定義「都沒按 = 0V」
                         │
                        GND
```
| 鈕 | 串聯電阻 | 按下 GPIO35 | ADC 約 |
|----|:--:|:--:|:--:|
| ＋ | 1kΩ | 3.0V | ~3720 |
| − | 4.7kΩ | 2.24V | ~2780 |
| 確定 | 10kΩ | 1.65V | ~2050 |
| 返回 | 22kΩ | 1.03V | ~1280 |
| 都沒按 | — | 0V | ~0 |

> 輕觸鈕 4 腳兩兩相通：一邊全接 3V3 共用軌，另一邊各接自己的串聯電阻，4 顆電阻另一端併成節點接 GPIO35。

### 肩鍵 L / R
每顆按鈕：一腳 → GPIO13 / 14，另一腳 → GND。按下接通（程式內部上拉，按下讀 LOW）。

### NRF24L01+PA/LNA
CE=4、CSN=5、SCK=18、MOSI=23、MISO=19；**VCC = 3V3（接 5V 會燒）**；**100µF 焊在 VCC-GND**。

### LCD 2004A（I²C）
SDA=21、SCL=22；**VCC = 5V**；GND → GND。

## 四、供電（2S 7.4V → buck → 5V）

```
2S 7.4V ─[電源開關]─→ buck(調 5.0V) ─┬→ ESP32 5V/VIN 腳
                                       └→ LCD VCC (5V)

ESP32 3V3 腳 ─┬→ NRF24 VCC (+100µF)
              ├→ 兩顆搖桿 VCC
              └→ 模式開關分壓 / 選單階梯 的 3V3 參考

所有 GND 共接。
```
ESP32 板上是 AMS1117-3.3（降 3.3V、只吃約 5V），**2S 不可直灌 VIN**，要先 buck 到 5V。
最省事替代：直接插 5V 行動電源走 USB，免 buck。

## 五、電阻總表（共 9 顆，1/4W）

| 阻值 | 數量 | 用在哪 |
|:--:|:--:|------|
| 1kΩ | 1 | 選單「＋」串聯 |
| 4.7kΩ | 1 | 選單「−」串聯 |
| 10kΩ | **6** | 選單「確定」串聯 ×1 + 選單節點下拉 ×1 + 開關 A ×2 + 開關 B ×2 |
| 22kΩ | 1 | 選單「返回」串聯 |

## 六、三條鐵則

1. **搖桿、NRF24 都吃 3V3，不能 5V**（接 5V：搖桿超壓打壞 ADC、NRF24 直接燒）。
2. **buck 先接電池量 OUT、轉到剛好 5.0V，再接 ESP32**（LM2596 出廠電壓亂的）。
3. **NRF24 的 100µF 電容不能省**（發射尖峰微秒級，靠它補，buck 來不及）。

## 七、燒錄後桌上驗證

| 驗證 | 怎麼看 / 怎麼修 |
|------|------|
| 油門方向 | 推到底 LCD 顯示 `Thr 0%`；反了改程式 `REV_THROTTLE` |
| 模式碼 | 撥開關，模式應為 上上=0 … 下下=8；不對改 `readSwitch3` 門檻 |
| 選單鈕 | 按各鈕認對；ADC 偏了改 `readMenuBtn` 門檻 |
| 肩鍵 yaw | 方向反了把 GPIO13 / 14 兩腳對調 |
| 雙向遙測 | LCD 出現飛機姿態/電量、`Link OK`（要飛機端同時開機） |

> ⚠️ 飛機端 `Drone_FC_PID` 與本手把封包已一起改成 9 模式，**兩支必須同時重燒**，否則收到亂碼。
