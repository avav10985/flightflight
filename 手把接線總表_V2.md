> 注意：V1（30-pin WROOM + 2004A LCD）已跳過，舊的 `手把接線總表.md` 不再對應實作。

## 〇、元件對元件接線

> **架構**(2026-06-03 最新版):A 板腦 + **C 板電阻中繼** + B 板前面板(含 TFT)。
> A↔C 用**杜邦線 18 條**(可拔);C↔B 用**主排線 12 條**(輸入訊號)+ **副排線 5 條**(TFT 訊號)。

### 螢幕(MSP2806 2.8" SPI ILI9341)↔ ESP32-S3

> MSP2806 **物理上裝在 B 板**(前面板),單排 14 隻排針插 B 板的 1×14 母排座。
> 訊號從 B 板出來經 **5 條副排線 → C 板 → A↔C 杜邦 → A 板**(SPI 部分跟 NRF24/SD 共用,在 C 板上分歧)。

| ESP32-S3 | MSP2806 排針 | 訊號路徑 |
|:--:|:--:|--|
| GPIO 48 | **CS** | A 板 GPIO 48 → A↔C 杜邦「TFT CS」→ C 板 → 副排線 #1 → B 板 TFT CS |
| GPIO 21 | **DC**(或 RS) | A 板 GPIO 21 → A↔C 杜邦「TFT DC」→ C 板 → 副排線 #2 → B 板 TFT DC |
| GPIO 38 | **SCK** | A 板 GPIO 38 → A↔C 杜邦「SCK」→ C 板(分歧:給 SD + 副排線 #3 → B 板 TFT SCK)|
| GPIO 39 | **SDI(MOSI)** | A 板 GPIO 39 → A↔C 杜邦「MOSI」→ C 板(分歧:給 SD + 副排線 #4 → B 板 TFT MOSI)|
| GPIO 40 | **SDO(MISO)** | A 板 GPIO 40 → A↔C 杜邦「MISO」→ C 板(分歧:給 SD + 副排線 #5 → B 板 TFT MISO)|

TFT 其他腳(在 B 板上**直接**接,不走副排線):

| MSP2806 排針 | 接到 | 說明 |
|:--:|--|--|
| **VCC** | B 板 3V3 軌(3V3 從 C↔B 主排線 #1 來) | 主電源 |
| **GND** | B 板 GND 軌 | — |
| **RESET** | B 板 3V3 軌(**直接接,不用 R10 上拉**)| 平常拉高 = 不重置 |
| **LED** | B 板 3V3 軌(背光常亮) | 想 PWM 控亮度 → 接 GPIO |
| **T_CLK / T_CS / T_DIN / T_DO / T_IRQ** | **不接** | 觸控版才用,V2-A 沒觸控 |

**省下來的 GPIO 11~18**(原本給並列資料的 8 隻)**全部釋出**——可給 V2-B、額外按鈕、LED 等。
**R10 變備用**:留 C 板上但無功能(MSP2806 不需要 RESET 上拉,直接拉 3V3 軌即可)。

### SD 卡 ↔ ESP32-S3(兩個選項,**擇一接**)

> 有兩個 SD 來源可選,**訊號相同、只能挑一個接 CS**:
>
> **A 案:MSP2806 板上的全尺寸 SD 座** —— 螢幕背面那個內建的座
>   - 優點:不用多焊一片模組
>   - 缺點:**全尺寸 SD 卡(32×24mm)會凸出來頂到搖桿**;microSD 要先插轉接卡
>
> **B 案:獨立 microSD 卡模組** —— C 板上多焊一片小模組
>   - 優點:**microSD 直接插**(免轉接卡)、模組小不會頂到任何東西
>   - 缺點:多焊 6 個焊點 + 模組佔 C 板空間

**共用的部分(兩個都這樣接)**:

| ESP32-S3 | SD 訊號 | 說明 |
|:--:|:--:|--|
| GPIO 38 | **SCK** | 跟 NRF24 / TFT 共用 |
| GPIO 39 | **MOSI** | 跟 NRF24 / TFT 共用 |
| GPIO 40 | **MISO** | 跟 NRF24 / TFT 共用 |
| 3V3 軌 | VCC | |
| GND 軌 | GND | |

**CS 線(擇一接到 GPIO 0)**:

| 你選的 | CS 接到 |
|--|--|
| A 案:用 MSP2806 內建 SD 座 | **MSP2806 的 SD_CS 腳** → GPIO 0(經 R11 10kΩ 上拉) |
| B 案:用獨立 microSD 模組 | **模組的 CS 腳** → GPIO 0(經 R11 10kΩ 上拉) |

⚠️ **絕對不要兩個 CS 都接到 GPIO 0** —— 同時兩個裝置會在同一個 CS 信號下打架,SD 讀寫亂掉。
⚠️ **沒選的那個保持空著**:選 A 就模組不接(或先佈線備用)、選 B 就 MSP2806 SD 座不插卡。

**建議的「先佈線、後決定」做法**:
1. C 板兩條 CS 線**都先預留接點**(GPIO 0 出來的線只焊一個端,另一個留空)
2. 兩個 SD 來源都接好其他訊號(SCK/MOSI/MISO/VCC/GND 反正都共用)
3. **試裝後決定**:全尺寸 SD 真的頂搖桿 → 跳線 GPIO 0 到模組 CS;不頂 → 跳線到 MSP2806 SD_CS

## 三、各元件接法細節

### 搖桿（2 顆）
- **左搖桿**:上下(Y) → **GPIO1**(油門,已拆彈簧);**左右(X) → GPIO10**(可選 yaw,彈簧留著)。
- **右搖桿**：上下(Y) → **GPIO2**（pitch）、左右(X) → **GPIO4**（roll）；彈簧留著。
- VCC → **3V3（不能 5V！）**、GND → GND、各軸抽頭 → 對應 ADC 腳。
- yaw 控制有兩個來源,**預設用肩鍵**(GPIO 8/9);程式選單可切換到左搖桿 X 軸(GPIO 10)。

### 模式開關 A / B（各一顆 MTS-103，3 腳）
```
上腳 ──────── 3V3
中腳(common) ─ GPIO（A=5 / B=6）
下腳 ──────── GND

節點(GPIO) 加分壓定義中檔：
節點 ──[10kΩ]── 3V3
節點 ──[10kΩ]── GND
```
撥上=0、中(斷)=1、下=2；**模式 = A×3 + B**（00=0 … 22=8）。

### 選單鈕電阻階梯（4 鈕共用 GPIO7）
```
3V3 ─┬─[＋鈕]──[1kΩ]───┐
     ├─[−鈕]──[4.7kΩ]──┤
     ├─[確定]──[10kΩ]───┼──→ GPIO7
     └─[返回]──[22kΩ]───┘
                         │
                       [10kΩ]   ← 下拉，定義「都沒按 = 0V」
                         │
                        GND
```
門檻與 V1 同（3250 / 2415 / 1665 / 640），程式內可實測微調。

### 肩鍵 L / R
每顆按鈕：一腳 → GPIO8 / 9，另一腳 → GND。按下接通（內部上拉，按下讀 LOW）。

### TFT MSP2806(2.8" SPI ILI9341,直立 240×320)— **物理在 B 板**

| MSP2806 排針 | 接到 | 路徑 |

## 四、供電（2S 7.4V → buck → 5V）

```
2S 7.4V ─[電源開關]─→ buck(調 5.0V) ─┬→ ESP32-S3 5V/VIN
                                       └→ (V2-B 加 MAX98357A 5V)

ESP32-S3 3V3 ─┬→ NRF24 VCC (+100µF)
              ├→ 經 A↔C 杜邦進 C 板 3V3 軌
              │   ├→ R6/R8/R11 上拉
              │   └→ 經 C↔B 主排線 #1 進 B 板 3V3 軌
              │       ├→ 兩顆搖桿 VCC
              │       ├→ MSP2806 TFT VCC + LED(背光)+ RESET(直接接,沒電阻)
              │       └→ 模式開關 / 選單階梯 的 3V3 參考
              └→ SD_CS 上拉(經 R11,SD 模組視位置在 B 或 C)

所有 GND 共接。
```

⚠️ **手把不做電池監測**（V2 設計決定，LCD 變暗自己會發現要充電）。

## 六、三條鐵則

1. **搖桿、NRF24 都吃 3V3，不能 5V**（搖桿超壓打壞 ADC、NRF24 直接燒）。
2. **buck 先接電池量 OUT、轉到剛好 5.0V，再接 ESP32**（LM2596 出廠電壓亂的）。
3. **NRF24 的 100µF 電容不能省**（發射尖峰微秒級，靠它補，buck 來不及）。

### 9.6 TFT MSP2806(鎖在外殼前面板,**插 B 板的 1×14 母排座**)

> MSP2806 排針**單排 14 隻 2.54mm**,直接對齊 B 板洞洞板,免轉接小板。
> **訊號 5 條走副排線到 C 板**(CS / DC / SCK / MOSI / MISO);電源/RESET/LED 在 B 板上自己接 3V3 軌。

| MSP2806 排針 | 接到(B 板上) |
|--|--|
| **CS** | C↔B 副排線 #1(→ C 板 → A↔C 杜邦 → S3 GPIO 48) |
| **DC**(或 RS) | C↔B 副排線 #2(→ C 板 → A↔C 杜邦 → S3 GPIO 21) |
| **RESET** | **B 板 3V3 軌**(直接接,**不用上拉電阻**)|
| **SDI(MOSI)** | C↔B 副排線 #4(→ C 板 SPI 分歧點 → 共用 SPI → S3 GPIO 39) |
| **SCK** | C↔B 副排線 #3(→ C 板 SPI 分歧點 → 共用 SPI → S3 GPIO 38) |
| **SDO(MISO)** | C↔B 副排線 #5(→ C 板 SPI 分歧點 → 共用 SPI → S3 GPIO 40;LovyanGFX init 用來讀 ID 確認晶片)|
| **LED**(背光) | **B 板 3V3 軌**(常亮;板上有限流電阻) |

| T_CLK / T_CS / T_DIN / T_DO / T_IRQ | **不接**(MSP2806 無觸控) |

### 9.7 SD 卡(兩個選項,**擇一接 CS**)

> 兩個來源都先佈線備用,**只接一個的 CS 到 GPIO 0**(兩個都接會打架)。

| SD 模組腳 | 接到 |

| MISO | GPIO 40(共用) |

### 9.12 選單 4 鈕電阻階梯(B 板)

4 顆鈕**共用「選單節點」**,各自串聯不同電阻:

| 按鈕 | 接腳 1 | 接腳 2 | 串聯電阻 |
|--|--|--|--|
| ＋ | 3V3 軌 | R1:1kΩ → 選單節點 | R1 = 1kΩ |
| − | 3V3 軌 | R2:4.7kΩ → 選單節點 | R2 = 4.7kΩ |
| OK | 3V3 軌 | R3:10kΩ → 選單節點 | R3 = 10kΩ |
| 返回 | 3V3 軌 | R4:22kΩ → 選單節點 | R4 = 22kΩ |

「選單節點」(在 C 板)還連:
- R5:10kΩ → GND 軌(下拉,定義「都沒按」)
- **A↔C 杜邦「選單節點」線(→ S3 GPIO 7)**

> 流向:＋/−/OK/返回 各自從 B 板經 B 排線 #9~#12 → C 板上的 R1~R4 → 匯流到「選單節點」→ 再一條 A↔C 杜邦線回 A 板 GPIO 7。

### 9.16 TFT MSP2806 在 B 板:訊號清單
