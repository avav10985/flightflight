> 焊接 + 買料用的單一對照表。對應程式：`Ground_TX_ESP32`（V2-A）。

## 〇、元件對元件接線

> **架構**(2026-06-02 確定):A 板腦 + **C 板被動元件樞紐** + B 板輸入。
> A↔C 用**杜邦線 17 條**(可拔,方便維修/換板);C↔B 用**排線 11 條**。
> **所有 11 顆電阻 + TFT + 肩鍵連接終點全部在 C 板**,B 板從此沒電阻。
> TFT 改用 **MSP2806 2.8" SPI 直立**,排針整齊對齊洞洞板,免做轉接小板。

### 螢幕(MSP2806 2.8" SPI ILI9341)↔ ESP32-S3

> MSP2806 是 **SPI 介面**,單排 14 隻排針(2.54mm 間距,直接對齊洞洞板)。
> SPI 訊號(SCK / MOSI / MISO)**跟 NRF24 和 SD 共用**,只佔 CS、DC 兩隻獨立腳。

| ESP32-S3 | MSP2806 排針(板上印的字) | 說明 |
|:--:|:--:|--|
| GPIO 48 | **CS** | TFT 晶片選(沿用原本 TFT_CS 那隻)|
| GPIO 21 | **DC**(或 RS) | data/command 切換(沿用原本 LCD_RS 那隻)|
| GPIO 38 | **SCK** | SPI 時脈(跟 NRF24 / SD 共用)|
| GPIO 39 | **SDI(MOSI)** | SPI 資料進 TFT(跟 NRF24 / SD 共用)|
| GPIO 40 | **SDO(MISO)** | SPI 資料出(跟 NRF24 / SD 共用,選用)|

TFT 其他腳:

| MSP2806 排針 | 接到 | 說明 |
|:--:|--|--|
| **VCC** | 3V3 軌(或 5V,看模組規格 — 多數 MSP2806 兩者皆可,優先用 3V3)| 主電源 |
| **GND** | GND 軌 | — |
| **RESET** | 3V3 軌(中間經 R10:10kΩ 上拉) | 平常拉高 = 不重置 |
| **LED** | 3V3 軌(背光常亮) | 想 PWM 控亮度 → 接 GPIO |
| **T_CLK / T_CS / T_DIN / T_DO / T_IRQ** | **不接** | 觸控版才用,V2-A 沒觸控 |

**省下來的 GPIO 11~18**(原本給並列資料的 8 隻)**全部釋出**——可給 V2-B、額外按鈕、LED 等。

## 三、各元件接法細節

### 搖桿（2 顆）
- **左搖桿**：只用上下(Y) → **GPIO1**（油門，已拆彈簧）；左右(X) 不用，空著。
- **右搖桿**：上下(Y) → **GPIO2**（pitch）、左右(X) → **GPIO4**（roll）；彈簧留著。
- VCC → **3V3（不能 5V！）**、GND → GND、各軸抽頭 → 對應 ADC 腳。

### 模式開關 A / B（各一顆 MTS-103，3 腳）
```
上腳 ──────── 3V3
中腳(common) ─ GPIO（A=5 / B=6）
下腳 ──────── GND

節點(GPIO) 加分壓定義中檔：
節點 ──[10kΩ]── 3V3
節點 ──[10kΩ]── GND
```
撥上=0、中(斷)=1、下=2；**模式 = A×3 + B**（00=0 … 22=8）。

### 選單鈕電阻階梯（4 鈕共用 GPIO7）
```
3V3 ─┬─[＋鈕]──[1kΩ]───┐
     ├─[−鈕]──[4.7kΩ]──┤
     ├─[確定]──[10kΩ]───┼──→ GPIO7
     └─[返回]──[22kΩ]───┘
                         │
                       [10kΩ]   ← 下拉，定義「都沒按 = 0V」
                         │
                        GND
```
門檻與 V1 同（3250 / 2415 / 1665 / 640），程式內可實測微調。

### 肩鍵 L / R
每顆按鈕：一腳 → GPIO8 / 9，另一腳 → GND。按下接通（內部上拉，按下讀 LOW）。

## 四、供電（2S 7.4V → buck → 5V）

```
2S 7.4V ─[電源開關]─→ buck(調 5.0V) ─┬→ ESP32-S3 5V/VIN
                                       └→ (V2-B 加 MAX98357A 5V)

ESP32-S3 3V3 ─┬→ NRF24 VCC (+100µF)
              ├→ 兩顆搖桿 VCC
              ├→ MSP2806 TFT VCC + LED(背光)+ RESET 上拉(經 R10)
              ├→ 模式開關 / 選單階梯 的 3V3 參考
              └→ SD_CS 上拉(經 R11)

所有 GND 共接。
```

⚠️ **手把不做電池監測**（V2 設計決定，LCD 變暗自己會發現要充電）。

## 六、三條鐵則

1. **搖桿、NRF24 都吃 3V3，不能 5V**（搖桿超壓打壞 ADC、NRF24 直接燒）。
2. **buck 先接電池量 OUT、轉到剛好 5.0V，再接 ESP32**（LM2596 出廠電壓亂的）。
3. **NRF24 的 100µF 電容不能省**（發射尖峰微秒級，靠它補，buck 來不及）。

|--|--|--|--|--|
| C1 | 100µF / ≥10V 電解 | NRF24 VCC ↔ GND | A 板(緊貼 NRF24) | 補發射尖峰,**不能省** |
| C2 | 0.1µF 陶瓷 | NRF24 VCC ↔ GND | A 板(跟 C1 並聯,選用) | 補高頻 |
| C3 | 0.1µF 陶瓷 | S3 VIN ↔ GND | A 板(緊貼 S3 電源腳,選用) | S3 高頻穩 |
| C4 | 0.1µF 陶瓷 | buck OUT+ ↔ GND | A 板(緊貼 buck 輸出,選用) | 補 buck 輸出 |

> C2/C3/C4 是「選用」——沒接也能跑,接了更穩。C1 是必接。

### 9.19 共享節點清單(同一個焊點要接多隻腳的)

| 節點名稱 | 位置 | 連到哪些 |
|--|--|--|
| 3V3 軌 | A 板頂邊 | S3 三個 3V3 腳、NRF24 VCC、兩顆搖桿 VCC、開關/階梯的 3V3 端,詳見下面 |
| 5V 軌 | A 板頂邊 | S3 5V 腳、TFT VCC、buck OUT+ |
| GND 軌 | A 板頂邊 | 所有元件的 GND |
| SW_A 節點 | B 板 | MTS-103 A 的 common、R6 一端、R7 一端、B 排線 #6 |
| SW_B 節點 | B 板 | MTS-103 B 的 common、R8 一端、R9 一端、B 排線 #7 |
| 選單節點 | B 板 | R1~R4 一端、R5 一端、B 排線 #8 |

**3V3 軌實際連到**:
- S3 左 #1、#2(兩個 3V3)
- S3 右 #22(3V3)
- NRF24 VCC + 100µF 一端
- 兩顆搖桿 VCC
- MTS-103 A/B 各自上腳
- 4 個選單鈕的 3V3 那腳
- R6、R8(SW 上拉)
- R10(LCD_RST 上拉)
- R11(SD_CS 上拉)
- TFT BL(背光)

> 看起來很多,實際就是「沿 A 板頂邊一條粗紅線,沿途分支接出去」。
