## 手把接線總表（V2-A：ESP32-S3-N16R8 + MAR2406 並列 TFT）

> 焊接 + 買料用的單一對照表。對應程式：`Ground_TX_ESP32`（V2-A）。
> 設計理由與全套規劃見 `手把V2_討論摘要.md`；本檔只列「買什麼、接哪裡」。
> 注意：V1（30-pin WROOM + 2004A LCD）已跳過，舊的 `手把接線總表.md` 不再對應實作。

## 〇、元件對元件接線

> **架構**(2026-06 確定):A 板腦 + **C 板被動元件樞紐** + B 板輸入。
> A↔C 用**杜邦線 26 條**(可拔,方便維修/換板);C↔B 用**排線 11 條**。
> **所有 11 顆電阻 + TFT + 肩鍵連接終點全部在 C 板**,B 板從此沒電阻。

### A 板(腦)上有什麼

| 元件 | 數量 | 備註 |
|--|:--:|--|
| ESP32-S3-N16R8 | 1 | 用 2×22 排母可拔 |
| NRF24L01+PA/LNA | 1 | 用排母可拔,緊貼右側 GPIO 38~42 |
| 100µF 電解電容 C1 | 1 | 焊在 NRF24 VCC↔GND(< 5mm)|
| 0.1µF 陶瓷電容 C2 / C3 / C4 | 3 | 選用:NRF24、S3、buck 出口各一 |

## 三、各元件接法細節

### 搖桿（2 顆）
- **左搖桿**：只用上下(Y) → **GPIO1**（油門，已拆彈簧）；左右(X) 不用，空著。
- **右搖桿**：上下(Y) → **GPIO2**（pitch）、左右(X) → **GPIO4**（roll）；彈簧留著。
- VCC → **3V3（不能 5V！）**、GND → GND、各軸抽頭 → 對應 ADC 腳。

### 模式開關 A / B（各一顆 MTS-103，3 腳）
```
上腳 ──────── 3V3
中腳(common) ─ GPIO（A=5 / B=6）
下腳 ──────── GND

節點(GPIO) 加分壓定義中檔：
節點 ──[10kΩ]── 3V3
節點 ──[10kΩ]── GND
```
撥上=0、中(斷)=1、下=2；**模式 = A×3 + B**（00=0 … 22=8）。

### 選單鈕電阻階梯（4 鈕共用 GPIO7）
```
3V3 ─┬─[＋鈕]──[1kΩ]───┐
     ├─[−鈕]──[4.7kΩ]──┤
     ├─[確定]──[10kΩ]───┼──→ GPIO7
     └─[返回]──[22kΩ]───┘
                         │
                       [10kΩ]   ← 下拉，定義「都沒按 = 0V」
                         │
                        GND
```
門檻與 V1 同（3250 / 2415 / 1665 / 640），程式內可實測微調。

### 肩鍵 L / R
每顆按鈕：一腳 → GPIO8 / 9，另一腳 → GND。按下接通（內部上拉，按下讀 LOW）。

## 四、供電（2S 7.4V → buck → 5V）

```
2S 7.4V ─[電源開關]─→ buck(調 5.0V) ─┬→ ESP32-S3 5V/VIN
                                       ├→ TFT VCC (5V)
                                       └→ （V2-B 加 MAX98357A 5V）

ESP32-S3 3V3 ─┬→ NRF24 VCC (+100µF)
              ├→ 兩顆搖桿 VCC
              ├→ 模式開關 / 選單階梯 的 3V3 參考
              └→ LCD_RST / LCD_RD 上拉

所有 GND 共接。
```

⚠️ **手把不做電池監測**（V2 設計決定，LCD 變暗自己會發現要充電）。

|:--:|:--:|------|
| 1kΩ | 1 | 選單「＋」串聯 |

| 22kΩ | 1 | 選單「返回」串聯 |

## 六、三條鐵則

1. **搖桿、NRF24 都吃 3V3，不能 5V**（搖桿超壓打壞 ADC、NRF24 直接燒）。
2. **buck 先接電池量 OUT、轉到剛好 5.0V，再接 ESP32**（LM2596 出廠電壓亂的）。
3. **NRF24 的 100µF 電容不能省**（發射尖峰微秒級，靠它補，buck 來不及）。
